# Portfolio

Minimal single-page portfolio for Rafael Balle.

## Tech stack

- **HTML/CSS/JavaScript only** – no build step, just static files
- **Optional local server**: you can run any static file server (for example `npx serve .`) to preview locally

## Structure

- `index.html` – main page layout
- `styles.css` – typography, layout and custom cursor styling
- `script.js` – animated cursor and project-card tilt interaction

## Running locally

Inside this folder:

```bash
npx serve .
```

Then open the shown URL in your browser.

## Customisation

- Replace placeholder text (name, intro, project descriptions, links) in `index.html`
- Adjust colors, spacing and typography in `styles.css`
- Tweak cursor feel and tilt strength in `script.js`
